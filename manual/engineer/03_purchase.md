# 購入フローマニュアル

## 前提条件

- MetaMaskがブラウザにインストール済みであること
- MetaMaskにPolygon Amoy Testnetが追加済みであること（`manual/01_setup.md` 参照）
- テスト用MATIC（ガス代）を保有していること
- テスト用JPYC（決済用）を保有していること

## 購入フロー概要

1. MetaMask接続
2. 商品選択
3. JPYC Approve（使用許可）
4. Buy実行（購入）
5. トランザクション確認

## 詳細手順

### 1. MetaMask接続

1. サイトにアクセスする
2. 「ウォレットを接続」ボタンをクリック
3. RainbowKitの接続モーダルが表示される
4. MetaMaskを選択
5. MetaMaskのポップアップで接続を承認

**注意:** ネットワークがPolygon Amoy Testnetになっていることを確認してください。異なるネットワークの場合、自動的にネットワーク切り替えが要求されます。

### 2. 商品選択

1. 商品一覧ページで購入したい商品を選択
2. 商品詳細ページで価格と分配設定を確認
3. 「購入」ボタンをクリック

### 3. JPYC Approve（使用許可）

購入ボタンをクリックすると、まず **Approve** トランザクションが実行されます。

1. MetaMaskのポップアップが表示される
2. 内容を確認:
   - トランザクションタイプ: ERC-20 トークン使用許可
   - 対象トークン: JPYC
   - 許可する使用量: 商品価格分のJPYC
   - 使用を許可する相手: 決済コントラクトアドレス
3. 「確認」をクリックしてトランザクションを送信
4. トランザクションの確認を待つ（数秒〜数十秒）

**Approveとは:** JPYCはERC-20トークンです。コントラクトがあなたのJPYCを引き出せるように、事前に「使用許可」を与える必要があります。これはERC-20の標準的な仕組みです。

### 4. Buy実行（購入）

Approveが完了すると、自動的に **Buy** トランザクションが実行されます。

1. MetaMaskのポップアップが再度表示される
2. 内容を確認:
   - トランザクションタイプ: コントラクト呼び出し（buy関数）
   - ガス代: 少量のMATIC
3. 「確認」をクリックしてトランザクションを送信
4. トランザクションの確認を待つ

Buy実行時にコントラクトが以下を行います:
- あなたのウォレットからJPYCを引き出し
- 設定された分配割合に従って各受取人にJPYCを即時送金

### 5. トランザクション確認

購入完了後、以下の方法でトランザクションを確認できます。

#### サイト上での確認

購入完了画面にトランザクションハッシュが表示されます。

#### PolygonScanでの確認

1. トランザクションハッシュをコピー
2. [Amoy PolygonScan](https://amoy.polygonscan.com/) にアクセス
3. 検索バーにトランザクションハッシュを貼り付けて検索
4. トランザクション詳細で以下を確認:
   - Status: Success
   - From: あなたのウォレットアドレス
   - Token Transfer: JPYC の移動が分配先ごとに記録されている

#### MetaMaskでの確認

MetaMaskの「アクティビティ」タブで最近のトランザクションを確認できます。

## エラーと対処法

### 「ガスが不足しています」

- MetaMask内のMATIC残高が不足しています
- [Polygon Faucet](https://faucet.polygon.technology/) からテスト用MATICを取得してください

### 「JPYCの残高が不足しています」

- 購入に必要なJPYC残高が不足しています
- テスト用JPYCを取得してください

### 「Approve failed」

- ネットワーク混雑やガス不足が考えられます
- 時間をおいて再度試してください

### 「Buy failed / Transaction reverted」

- Approveが完了していない可能性があります
- コントラクトの商品がアクティブでない可能性があります
- MetaMaskでApproveトランザクションが成功しているか確認してください

### トランザクションが長時間保留中

- Polygon Amoyネットワークが混雑している可能性があります
- MetaMaskでトランザクションの「スピードアップ」を試してください

## よくある質問

**Q: ガス代はいくらかかりますか？**
A: Polygonネットワークのガス代は非常に安く、通常1回のトランザクションで0.01 MATIC未満です。

**Q: 購入後のキャンセルはできますか？**
A: ブロックチェーン上のトランザクションは不可逆です。購入完了後のキャンセルや返金はできません。

**Q: 分配はリアルタイムで行われますか？**
A: はい。Buyトランザクションが成功した時点で、コントラクト内で即座に各受取人へJPYCが分配されます。
