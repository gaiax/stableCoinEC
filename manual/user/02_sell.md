# 出品ガイド

このガイドでは、StableCoinEC のダッシュボードから商品を登録・出品する手順を説明します。

---

## 前提条件

- 運営からショップアカウントが発行されていること
- ダッシュボードへのアクセス権があること（`/dashboard` ページ）

---

## ダッシュボードへのアクセス

ブラウザで以下の URL にアクセスしてください。

```
http://[サイトURL]/dashboard
```

---

## 商品を登録する

### 1. 商品情報を入力する

ダッシュボードの左側にある「**商品を登録する**」フォームに情報を入力します。

| 項目 | 説明 | 必須 |
|------|------|------|
| 商品名 | 購入者に表示される商品のタイトル | ✅ |
| 説明 | 商品の詳細説明（改行・詳細を自由に記載） | 任意 |
| 画像 URL | 商品画像の URL（https://... 形式） | 任意 |
| 価格（JPYC） | JPYC 建ての販売価格（例: 1000） | ✅ |

### 2. 売上の分配先を設定する

購入代金をどのアドレスに何 ％ 配分するかを設定します。

**設定方法:**

- 「**受取アドレス**」欄に分配先のウォレットアドレス（`0x`から始まる文字列）を入力します。
- 「**分配割合**」欄にベーシスポイント（BP）で割合を入力します。

**ベーシスポイントの計算:**

| 表示割合 | 入力値（BP） |
|---------|------------|
| 100 % | 10000 |
| 80 % | 8000 |
| 50 % | 5000 |
| 20 % | 2000 |
| 10 % | 1000 |

> **重要:** 全ての分配先の割合を合計すると **ちょうど 10000（= 100 %）** になる必要があります。合計が 10000 でない場合、登録ボタンを押してもエラーになります。

**例: 80 % / 20 % の 2 者分配**

| 受取アドレス | 分配割合（BP） |
|------------|-------------|
| `0xAAAA...` | 8000 |
| `0xBBBB...` | 2000 |
| **合計** | **10000 ✅** |

### 3. 「登録する」ボタンをクリック

入力内容を確認したら **「登録する」** ボタンをクリックします。

登録処理には **10〜30 秒程度** かかります。これはスマートコントラクトへのトランザクション送信を行っているためです。ページを閉じずにお待ちください。

### 4. 登録完了の確認

登録が完了すると、ダッシュボード右側の「**登録商品と注文**」欄に商品が表示されます。

---

## 商品の確認

登録した商品はトップページ（`/`）の商品一覧にも表示されます。

商品詳細ページ（`/products/[商品ID]`）では購入者と同じ画面を確認できます。

---

## 注文の確認

ダッシュボード右側の「**登録商品と注文**」欄で、各商品の注文数を確認できます。

---

## よくある質問

### Q. 登録後に商品情報を変更できますか？
A. 現時点では商品情報の変更機能はありません。価格や分配設定を変更したい場合は、新しい商品として再登録してください。オンチェーン情報（価格・分配先）は変更できないため、既存の商品を無効化したい場合は運営にご連絡ください。

### Q. 登録ボタンを押しても「Unauthorized」エラーが出ます。
A. ダッシュボードへのアクセス権が正しく設定されていない可能性があります。運営にご連絡ください。

### Q. 登録に時間がかかりすぎます。
A. Polygon ネットワークが混雑している場合、トランザクションの処理に時間がかかることがあります。数分待ってもページが更新されない場合は、ダッシュボードを再読み込みして商品一覧を確認してください。

### Q. 商品画像はどこにアップロードしますか？
A. 現時点では画像 URL を直接入力する形式です。画像は外部サービス（Google Drive の公開リンク、Imgur、など）にアップロードして URL を取得してください。

### Q. 売上の受取はどうなりますか？
A. 購入者が購入を完了した瞬間、スマートコントラクトが自動的に設定した分配割合で各受取アドレスに JPYC を送金します。売上の受け取りに追加の操作は不要です。

## トラブルシューティング

| 問題 | 対処法 |
|------|--------|
| 「Missing required fields」エラー | 必須項目（商品名・価格・分配設定）をすべて入力してください |
| 「Split percentages must total 10000」エラー | 分配割合の合計が 10000 になるよう調整してください |
| 「Internal server error」エラー | 時間をおいて再度試してください。改善しない場合は運営にご連絡ください |
| 登録後も商品一覧に表示されない | ページを再読み込みしてください |
